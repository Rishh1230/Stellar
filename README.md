#  Stellar — Real-Time IoT Data Platform

A high-performance, real-time IoT data platform designed for ingesting, processing, and analyzing high-frequency sensor data streams with low latency.

---

##  Overview

Stellar is a full-stack system that enables real-time data acquisition from IoT devices, processes streaming data efficiently, and provides predictive insights using machine learning models.

The system is built with a focus on:

* Low-latency data handling
* Scalable backend architecture
* Real-time analytics and monitoring

---

##  Key Features

* **Real-Time Data Ingestion**
  Handles continuous sensor data streams efficiently

*  **Scalable Backend APIs**
  Built using Node.js and Express for high-throughput processing

*  **Time-Series Data Processing**
  Processes and analyzes streaming data in real time

* **Anomaly Detection & Prediction**
  Integrates ML models for pattern recognition and forecasting

*  **Live Monitoring Interface**
  Frontend dashboard for real-time visualization

---

### Backend

### Frontend

### Data & ML

* Python
* NumPy, Pandas

### Database

* MongoDB

##  System Design Highlights

* Designed to handle **high-frequency event streams**
* Modular API architecture for scalability
* Separation of concerns between ingestion, processing, and visualization
* Optimized for **low-latency data flow**

##  Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Rishh1230/stellar.git
cd stellar
```

### 2. Install dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

### 3. Run the application

#### Start backend

```bash
npm start
```

#### Start frontend

```bash
npm start
```
